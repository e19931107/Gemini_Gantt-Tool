# 硬體產品開發專案甘特圖 (Gantt Chart for Hardware Product Development)

這是一個基於 HTML/Tailwind CSS/Vanilla JS 實現的互動式甘特圖，用於追蹤硬體產品開發（Pre-EVT, EVT, DVT, PVT, MP）的專案進度。

## ⚡️ 演示 (Demo)

您可以點擊以下連結，在瀏覽器中直接運行和互動：

[**👉 點擊這裡運行甘特圖**](gantt-chart-demo.html)

**功能特色：**

* 自動計算專案時間軸
* 任務欄顏色區分不同開發階段 (EVT, DVT, PVT, MP)
* 任務新增、編輯和刪除功能
* 響應式設計 (RWD) 兼容
* 日期單元格顯示週末和今日標記

---

## 🛠️ 技術棧

* **HTML5**
* **Tailwind CSS** (透過 CDN 載入)
* **Vanilla JavaScript** (原生 JS)
